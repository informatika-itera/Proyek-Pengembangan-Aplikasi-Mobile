# 💰 TabungIn - KMP Project

Aplikasi **manajemen target tabungan** berbasis mobile multiplatform untuk mata kuliah **Pengembangan Aplikasi Mobile** di ITERA.

TabungIn memfasilitasi pengguna untuk mencatat dan memantau progres celengan fisik secara digital, dilengkapi asisten finansial berbasis AI untuk membantu perencanaan tabungan yang lebih cerdas.

## ✨ Fitur Aplikasi

- 💰 **CRUD Target Tabungan** — Tambah, edit, hapus, dan pantau target tabungan
- 💵 **Catat Setoran** — Rekam setiap setoran ke target tertentu
- 📊 **Progress Visual** — Progress bar reaktif menampilkan persentase pencapaian
- 🤖 **AI Asisten Finansial** — Tips menabung dan saran perencanaan keuangan via Gemini API
- 🌙 **Dark Mode** — Tema gelap/terang
- 📱 **Cross-Platform** — Android & iOS dari satu codebase

### Clean Architecture + MVVM

```

                                    │
┌───────────────────────────────────┼─────────────────────────┐
│                      DOMAIN LAYER │                          │
│                    ┌──────────────▼──────────────┐          │
│                    │         Use Cases           │          │
│                    │    (Business Logic)         │          │
│                    └──────────────┬──────────────┘          │
│                    ┌──────────────▼──────────────┐          │
│                    │    Repository Interface     │          │
│                    └──────────────┬──────────────┘          │
└───────────────────────────────────┼─────────────────────────┘
                                    │
┌───────────────────────────────────┼─────────────────────────┐
│                       DATA LAYER  │                          │
│                    ┌──────────────▼──────────────┐          │

│                    └──────────────┬──────────────┘          │
│              ┌────────────────────┼────────────────────┐    │
```

## 🚀 Getting Started

### Prerequisites

- Android Studio Ladybug (2024.2.1) atau lebih baru
- JDK 17+
- Xcode 15+ (untuk iOS, opsional)

### Setup

1. **Clone repository**
   ```bash
   git clone https://github.com/informatika-itera/Proyek-Pengembangan-Aplikasi-Mobile.git
   cd Proyek-Pengembangan-Aplikasi-Mobile

   # Checkout ke branch project kelompok
   git checkout project/123140129-123140143-TabungIn
   ```

2. **Setup `local.properties`**
   ```bash
   cp local.properties.example local.properties
   # Edit local.properties, isi GEMINI_API_KEY
   ```

3. **Sync & Build**
   ```bash
   ./gradlew build
   ./gradlew :composeApp:assembleDebug
   ```

4. **Run**
   - **Android**: pilih run configuration `composeApp` di Android Studio, atau
     ```bash
     ./gradlew :composeApp:installDebug
     ```

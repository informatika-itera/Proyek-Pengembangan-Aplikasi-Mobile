## ✨ Tentang Aplikasi

Feelia adalah aplikasi jurnal harian yang mencatat emosi pengguna, lalu menganalisis pola dan tren emosi tersebut menggunakan AI

### Fitur Unggulan
#### Core Features
Tulis Jurnal Harian → Input teks singkat 1-2 kalimat tentang perasaan hari ini
Deteksi Emosi Otomatis → Gemini AI analisis teks dan kasih label emosi (senang, sedih, cemas, marah, netral)
Riwayat Jurnal → Lihat semua jurnal yang pernah ditulis
Edit & Hapus Jurnal → CRUD lengkap

#### Data & Analytics Features
Grafik Tren Mood → Visualisasi emosi harian dan mingguan
Statistik Emosi → Emosi apa yang paling sering muncul minggu ini
Insight AI → Gemini kasih insight seperti "Kamu sering cemas di hari Senin"
Kata yang Sering Muncul → Deteksi kata dominan saat mood tertentu

#### Supporting Features
Search Jurnal → Cari jurnal berdasarkan kata kunci
Filter by Mood → Filter jurnal berdasarkan emosi tertentu
Dark Mode → Sudah ada di template, tinggal pakai
kalau dari AI ini katanya

# Aplikasi Kasir Digital UMKM

Aplikasi ini berbasis Android yang dirancang untuk membantu pelaku UMKM dalam mengelola transaksi penjualan secara digital, cepat, dan efisien.

## Fitur Utama
- Dashboard Penjualan
- Manajemen Produk
- Tambah, Edit, dan Hapus Produk
- Pencarian Produk
- Transaksi Penjualan
- Perhitungan Total Otomatis
- Perhitungan Kembalian Otomatis
- Riwayat Transaksi
- Laporan Penjualan
- Penyimpanan Data Produk dan Transaksi
- Tampilan UI Modern dan Responsif

## Teknologi
- Android Studio
- Kotlin / Java
- SQLite / Room Database
- Material Design UI

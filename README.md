---

##  Fitur Aplikasi

-  **CRUD Transaksi** — Tambah, edit, hapus, dan lihat catatan pengeluaran/pemasukan
- ️ **Kategori Budget** — Kategorikan transaksi (makan, transportasi, belanja, dll)
-  **Statistik Keuangan** — Grafik pengeluaran per kategori per bulan
-  **Riwayat Transaksi** — Lihat semua catatan dengan filter tanggal
-  **Dark Mode** — Tema gelap/terang

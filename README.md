## Tentang Aplikasi
 
**StudyPlanner** adalah aplikasi mobile untuk membantu mahasiswa merencanakan dan mengelola kegiatan belajar sehari-hari. Pengguna dapat mencatat tugas per mata kuliah, mengatur jadwal belajar, dan memantau progres, semuanya dalam satu aplikasi.
 
Aplikasi ini dibangun di atas template **NoteAI KMP** menggunakan **Kotlin Multiplatform** sehingga berjalan di Android maupun iOS dari satu codebase, dengan arsitektur **Clean Architecture + MVVM** dan bantuan AI dari Google Gemini.
 
> Proyek ini masih dalam tahap awal pengembangan.
